dcolor
======

Minimal script to dynamically theme terminals.

## Usage

**Set a default theme**
```
echo "eink" > /path/to/dcolor/current
```
**Set theme**
```
/path/to/dcolor/dcolor
```

or-

```
/path/to/dcolor/dcolor nord
```
